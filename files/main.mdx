## Overview

The `main.py` file serves as the entry point for the AGI Experiment application. It's a simple launcher that delegates to the CLI module.

## Code Breakdown

```python main.py
#!/usr/bin/env python3
"""
AGI Experiment - Autonomous Social Media Agent
Main entry point
"""

from ollie.cli import main

if __name__ == "__main__":
    main()
```

## What It Does

1. **Shebang Line**: `#!/usr/bin/env python3` makes the file executable as a script
2. **Import**: Imports the main function from the CLI module
3. **Entry Point**: Calls the CLI main function when run directly

## Usage

### Direct Execution

```bash
python main.py
```

### Module Execution

```bash
python -m ollie.cli
```

Both methods achieve the same result - launching the CLI interface.
